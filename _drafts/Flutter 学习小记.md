## Dart

####  Futures

Dart 与 JavaScript 类似，同样是单线程模型。在 JavaScript 中，Promise 对象代表异步操作的完成或者失败。

Dart 使用 [`Future`](https://dart.cn/tutorials/language/futures) 对象来实现该机制。
